The Outpost Broadcast Real-Time NodeJS Chat Application

Demo Stack

You can put a stack on any Smallest Federated Wiki page, and all of the content broadcast on each individual stack is independent from the others. This app is run along side a SFW farm, and your username is associated with all of the stacks created within a farm.

About

This web application uses socket.io to create full-duplex communication between the client and the server. All data is stored as .json files on the server and is loaded into memory by NodeJS. This demo is a stand-alone Broadcast, but it is designed to be used on The Outpost as a Smallest Federated Wiki plug-in.
