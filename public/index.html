Broadcast.TheOutpost.io Real-Time NodeJS Chat Application

The Concept

Your browser can send any data of any form to the server, and the server does not necessarily need to keep it. It can simply be a middle man that relays data you send out to everyone currenltly connected. It has a new role: instead of archiving information, it broadcasts information.

Imagine how cool this will be when you can send files...

Try it out! Click on the text box and leave a message. See people type as they go...

About

This web application uses socket.io to create full-duplex communication between the client and the server. All data is stored as .json files on the server and is loaded into memory by NodeJS. Your typing is simply relayed to other connected users, but once you hit enter, it is recorded by the server.

How It Works

You can put a stack on any Smallest Federated Wiki page, and all of the content broadcast on each individual stack is independent from the others. This app is run along side a SFW farm, and your username is associated with all of the stacks created within a farm.

On the other hand, you can simply put this on any web server. This demo here is stand-alone. You can integrate it into your page, or you can have an iframe that refers to another server.
